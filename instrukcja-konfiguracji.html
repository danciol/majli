Instrukcja konfiguracji serwisów zewnętrznych
Krok po kroku jak uruchomić wszystkie niezbędne usługi przed pierwszym uruchomieniem aplikacji.

Spis treści

  1. Firebase — baza danych i logowanie wymagane
  2. Cloudinary — galeria zdjęć darmowe
  3. TextBee — wysyłanie SMS darmowe
1
Firebase
Wymagane

Firebase to baza danych i system logowania aplikacji. Bez niego nic nie działa.

1
Wejdź na console.firebase.google.com i zaloguj się kontem Google.
2
Kliknij Utwórz projekt → wpisz nazwę (np. nazwa-salonu) → wyłącz Google Analytics → kliknij Utwórz projekt.
3
Dodaj aplikację webową: w projekcie kliknij ikonę </> → wpisz nazwę aplikacji → kliknij Zarejestruj. Skopiuj obiekt firebaseConfig — będzie potrzebny w kroku 5.
4
Włącz Firestore: w lewym menu → Build → Firestore Database → Create database → wybierz production mode → wybierz region (np. europe-west1) → kliknij Done.
5
Ustaw reguły Firestore: zakładka Rules → wklej poniższe reguły → kliknij Publish.
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /settings/{doc} { allow read: if true; allow write: if request.auth != null; } match /gallery/{doc} { allow read: if true; allow write: if request.auth != null; } match /{document=**} { allow read, write: if request.auth != null; } } }
6
Włącz Authentication: w lewym menu → Build → Authentication → Get started → zakładka Sign-in method → włącz Email/Password → Save.
7
Dodaj pierwsze konto admina: zakładka Users → Add user → wpisz email i hasło → Add user.
8
Dodaj pracownika do Firestore: w Firestore → Start collection → ID kolekcji: employees → ID dokumentu: (auto) → dodaj pola: name (string), email (string, ten sam co w Auth), role (string: admin).
Darmowy plan (Spark)
Firestore daje 1 GB miejsca, 50 000 odczytów i 20 000 zapisów dziennie — dla salonu wystarczy na lata bez płacenia.
3
Cloudinary — galeria zdjęć
Darmowe

Cloudinary przechowuje zdjęcia w galerii. Wgrywasz przez panel admina — pojawiają się od razu na stronie.

1
Wejdź na cloudinary.com → Sign Up Free → zarejestruj się (można przez Google).
2
Po zalogowaniu w dashboardzie znajdź i skopiuj swój Cloud Name (np. dxyz1234abc).
3
Przejdź do Settings → Upload → Upload presets → Add upload preset.
4
Ustaw: Preset name = gallery_unsigned, Signing Mode = Unsigned, opcjonalnie Folder = nazwa salonu. Zapisz.
5
W panelu admina → Ustawienia → Cloudinary → wpisz Cloud Name i Upload Preset → kliknij Zapisz.
Limity darmowego konta
25 GB miejsca + 25 GB transferu miesięcznie. Wystarczy na setki zdjęć salonu bez żadnych opłat.
4
TextBee — wysyłanie SMS
Darmowe

TextBee używa Twojego telefonu z Androidem jako bramki SMS. Wiadomości idą z Twojego numeru telefonu.

Wymagania
Potrzebujesz telefonu z Androidem (stary/zapasowy wystarczy), który będzie cały czas podłączony do prądu i internetu.
1
Wejdź na textbee.dev → zarejestruj konto.
2
Na telefonie z Androidem pobierz aplikację TextBee z Google Play.
3
Zaloguj się w aplikacji na telefonie tym samym kontem co na stronie.
4
W panelu na textbee.dev → Devices skopiuj Device ID swojego telefonu.
5
W textbee.dev → API Keys wygeneruj nowy klucz i skopiuj go.
6
W panelu admina → Ustawienia → TextBee SMS → wpisz API Key i Device ID → kliknij Zapisz.
Bez konfiguracji TextBee
Jeśli TextBee nie jest skonfigurowany, kliknięcie „Wyślij przypomnienie" otwiera aplikację SMS w telefonie z gotową wiadomością — możesz wysłać ręcznie.

Checklista przed uruchomieniem

☐ Firebase — projekt utworzony, Firestore i Auth włączone, reguły ustawione
☐ Firebase — pierwsze konto admina dodane w Authentication i Firestore
☐ Cloudinary — konto założone, Upload Preset (unsigned) utworzony
☐ Cloudinary — dane wpisane w panelu admina → Ustawienia
☐ TextBee — konto założone, aplikacja na Androidzie zainstalowana (opcjonalnie)
☐ TextBee — API Key i Device ID wpisane w panelu admina → Ustawienia (opcjonalnie)
